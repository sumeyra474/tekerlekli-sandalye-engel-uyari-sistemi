# tekerlekli-sandalye-engel-uyari-sistemi
Ultrasonik mesafe sensörü ve titreşim motoru kullanılarak tekerlekli sandalye kullanıcıları için geliştirilen engel uyarı sistemi.

Bu projede HC-SR04 ultrasonik mesafe sensörü
kullanılarak belirli bir mesafenin altına
girildiğinde titreşim motoru çalıştırılmıştır.

## Kullanılan Donanımlar
- Arduino / Deneyap Kart
- HC-SR04 Ultrasonik Sensör
- Titreşim Motoru
- Breadboard
- Jumper kablo 
## Çalışma Mantığı
Mesafe 20 cm altına düştüğünde motor aktif olur.
